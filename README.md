# Termux Minecraft Bot (BonGuard)

Бот на Node.js для Minecraft серверов (например `Bon4ikMines.aternos.me:24474`),
запускаемый прямо с телефона через **Termux**.

Возможности:
- Подключение к любому серверу в формате `айпи:порт` (Aternos поддерживается).
- Команды из терминала (русский язык): сменить ник, сменить сервер, сказать в чат,
  атаковать игрока, ходить за игроком, остановиться, и т.д.
- Режим **«Слушать только <Ник>»** — бот слушается ТОЛЬКО указанного хозяина.
  При этом остальных игроков он продолжает модерировать по 18 правилам сервера.
- Авто-реакция в чате на нарушения (нецензурщина, ссылки, флуд, продажа
  аккаунтов, пропаганда, угрозы взлома, ложные жалобы, оскорбление админов и
  т.д.) с упоминанием конкретного правила.
- PvP-самозащита: если по боту бьют — отвечает. Также периодически атакует
  ближайших враждебных мобов.
- Авто-переподключение при разрыве (полезно для Aternos, который иногда
  выкидывает игроков).

> ВАЖНО: бот не может реально банить или мутить игроков — у него нет прав
> администратора. Он только публично предупреждает нарушителей в чате с
> кулдауном (по умолчанию 30 секунд на одного игрока), указывая правило.
> Реальные наказания должен выдавать админ сервера.

## Установка в Termux

1. Установи [Termux из F-Droid](https://f-droid.org/en/packages/com.termux/)
   (версия из Google Play устарела и не работает).

2. Открой Termux и выполни:

   ```bash
   pkg update -y && pkg upgrade -y
   pkg install -y nodejs git
   ```

3. Скопируй папку `termux-bot` на телефон. Например, через `git`:

   ```bash
   git clone <твой-репозиторий> termux-bot
   cd termux-bot
   ```

   или вручную скинь файлы (`bot.js`, `rules.js`, `package.json`,
   `config.example.json`) в папку `~/termux-bot` через файловый менеджер.

4. Установи зависимости:

   ```bash
   npm install
   ```

5. Создай свой конфиг:

   ```bash
   cp config.example.json config.json
   nano config.json
   ```

   Минимально замени:
   - `"server"` — `айпи:порт` твоего сервера.
   - `"username"` — ник бота.
   - `"owner"` — твой ник в Minecraft (если хочешь сразу режим хозяина).
   - `"ownerOnlyMode": true` — если хочешь, чтобы бот сразу слушал только тебя.

6. Запусти:

   ```bash
   node bot.js
   ```

   Можно передать сервер и ник аргументами:

   ```bash
   node bot.js Bon4ikMines.aternos.me:24474 BonGuard
   ```

## Команды в терминале

Печатаешь прямо в окне Termux после строки `>`:

| Команда                       | Что делает                                              |
|-------------------------------|---------------------------------------------------------|
| `Слушать только <Ник>`        | Включить режим хозяина (бот слушает только этого ника). |
| `Слушать всех`                | Выключить режим хозяина.                                |
| `скажи <текст>`               | Отправить сообщение в чат сервера.                      |
| `атакуй <Ник>`                | Напасть на игрока.                                      |
| `иди <Ник>`                   | Ходить за игроком.                                      |
| `стой`                        | Остановиться (и драться, и ходить).                     |
| `ютуберы`                     | Список игроков с «ютуберкой» (им можно ссылки).         |
| `ютубер+ <Ник>`               | Добавить ютубера.                                       |
| `ютубер- <Ник>`               | Убрать ютубера.                                         |
| `сервер <айпи:порт>`          | Сменить сервер и переподключиться.                      |
| `ник <Ник>`                   | Сменить ник и переподключиться.                         |
| `правила`                     | Показать список правил сервера.                         |
| `стоп бот` / `выход`          | Выключить бота.                                         |
| `помощь`                      | Список команд.                                          |

## Команды в чате (только от хозяина)

Когда включён режим «Слушать только <Ник>» — хозяин может писать в чат сервера:

- `скажи <текст>` — бот скажет это в чат.
- `иди ко мне` — бот пойдёт за тобой.
- `стой` — остановится.
- `атакуй <Ник>` — нападёт на игрока.
- `правила` — пришлёт правила в чат.

Любой другой текст от хозяина бот просто повторит в чат (как «озвучка»).

## Правила сервера, которые бот контролирует

См. `rules.js`. По умолчанию реагирует на:

1. Читерство, 2. Продажа акков, 3. Флуд, 4. Ссылки/пиар, 5. Терроризм,
6. Уклон от проверки, 7. Бессмысленные сообщения, 9. Нецензурщина,
12. Оскорбление админов, 13. Ложные жалобы, 14. Краш/взлом сервера,
15. Взлом аккаунта, 16. Рейд, 17. Дюп, 18. Превышение полномочий админа.

Хочешь добавить или изменить слова-триггеры — открой `rules.js` и поправь
массивы (`PROFANITY`, `URL_REGEX`, `TERROR_PATTERNS` и т.д.).

## Чтобы Termux не засыпал

Чтобы бот не отключался когда экран потухнет:

```bash
termux-wake-lock
```

Снять блокировку:

```bash
termux-wake-unlock
```

## Известные ограничения

- Бот использует `auth: "offline"` — это нужно для пиратских/Aternos серверов.
  Для лицензионных серверов поставь `"auth": "microsoft"` в `config.json` —
  при первом запуске Microsoft попросит залогиниться.
- На Aternos сервер часто перезапускается и порт меняется. Узнай новый порт
  на сайте Aternos и применяй командой `сервер Bon4ikMines.aternos.me:НОВЫЙПОРТ`.
- У бота нет прав админа — он только пишет предупреждения в чат.
